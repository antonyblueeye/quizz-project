# Раунд 3 — угадай песню

Положите сюда 10 аудиофрагментов с именами:

- `round_3_1.mp3`
- `round_3_2.mp3`
- …
- `round_3_10.mp3`

Поддерживаются форматы, которые играет браузер: **MP3**, OGG, WAV, M4A.

Ответы (название песни и/или исполнитель) — в `app/data/quiz.json`, секция `"id": 3`.
Поле `accept` — дополнительные варианты, которые тоже засчитываются.
